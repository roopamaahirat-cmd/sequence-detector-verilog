# Sequence Detector using Verilog HDL

## Description
This project detects a specific binary sequence using Finite State Machine (FSM).

## Features
- Detects input sequence
- FSM based design
- Outputs signal when sequence matches

## Tools Used
- Vivado Design Suite

## Files Included
- sequence_detector.v
- sequence_detector_tb.v
